# feature-selection-knn-pso
Sınıflandırma performansını artırmak amacıyla Parçacık Sürü Optimizasyonu (PSO) ile optimize edilmiş KNN sınıflandırıcısı.
Bu proje, K-En Yakın Komşu (KNN) algoritması ile Parçacık Sürü Optimizasyonu (PSO) yöntemini entegre eden hibrit bir makine öğrenmesi yaklaşımı sunmaktadır. PSO algoritması, özellik seçimi ve hiperparametre optimizasyonu için kullanılarak sınıflandırma performansı artırılmıştır. Deneysel sonuçlar, standart KNN modeline kıyasla daha yüksek doğruluk elde edildiğini göstermektedir.
